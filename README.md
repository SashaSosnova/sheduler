# Каникулы — план и зарядка

Мобильное приложение для каникул: гибкий план дня, календарь и зарядка от миотерапевта с таймерами.

## Запуск

```bash
npm install
npm run dev
```

Открой адрес из терминала на компьютере. На телефоне или планшете в той же Wi‑Fi сети открой `http://<IP-компьютера>:5173`.

## Установка на телефон

1. Открой приложение в Chrome (Android) или Safari (iPhone/iPad).
2. Меню → **«На экран Домой»** / **Install app**.
3. Ярлык откроет приложение как обычную программу.

Данные хранятся на устройстве (localStorage).

## Экраны

- **Сегодня** — тип дня, обязательный минимум, заметка
- **Календарь** — месяц и отметки по дням
- **Зарядка** — список упражнений, таймеры, чекбоксы

Список упражнений можно редактировать: добавь свои из зарядки миотерапевта.
